# Post-study Work Visa

After completing study at a New Zealand education provider, you can apply for a Post-study Work Visa to work legally in New Zealand. This guide gives an overview of required materials and the application process.

::: tip
The following information is for reference only. Follow the latest requirements on the Immigration New Zealand website [immigration.govt.nz](https://www.immigration.govt.nz/).
:::

## Application Deadline

- You must submit the application **within 3 months after your Student Visa expires**
- PhD graduates may apply within 6 months

## Material Checklist

| Material | Description | Details |
|----------|-------------|---------|
| Passport | Personal information page | - |
| Medical report | eMedical medical exam (chest X-ray + possibly a full medical exam) | [Medical exam](/en/visa/work-visas/post-study-work-visa/medical-examination/) |
| Completion Letter | Course completion proof issued by the school | [Completion Letter](/en/visa/work-visas/post-study-work-visa/completion-letter/) |
| Transcript | Academic Transcript | - |
| Photo | ID photo meeting Immigration New Zealand requirements | - |
| Visa card | Pay the visa fee | - |

## Application Process Overview

1. **Medical exam**: visit an Immigration New Zealand-designated medical provider to complete the eMedical exam
2. **Obtain Completion Letter**: apply to your school for proof of course completion
3. **Apply online**: log in to the Immigration New Zealand system, complete and submit the work visa application

See [Immigration New Zealand online application process](/en/visa/work-visas/post-study-work-visa/immigration-application/).
