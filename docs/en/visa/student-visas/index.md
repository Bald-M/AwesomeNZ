# Student Visas

This section explains the application process and material preparation for New Zealand Student Visas.

## Covered Visa Types

| Type | Description | Link |
|------|-------------|------|
| [Student Visa](/en/visa/student-visas/student-visa/) | Visa for studying in New Zealand | Online application |

## Application Entry

Immigration New Zealand website: [immigration.govt.nz](https://www.immigration.govt.nz/)

- Go to "Explore and select a visa"
- Select "Student visa"
- Log in with a **RealMe** account and apply online

## Related Guides

Before applying for a Student Visa, you usually also need to prepare:

- [Pre-departure medical exam - chest X-ray](/en/pre-departure-medical/): chest X-ray valid for 36 months
- [Criminal record certificate](/en/criminal-record/): China / New Zealand
- [Bank statements](/en/bank-statement/): proof of funds
