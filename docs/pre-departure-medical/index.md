# 出国体检 · 胸片

申请签证时通常需要提供指定医院的出国体检胸片报告。本文介绍体检的材料准备、流程及注意事项。

::: tip
以下信息供参考，建议出行前致电医院确认最新要求及费用。
:::

## 材料准备

- **2 张 2 寸白底照片**（现场可能不需要，建议携带备用）
- **护照**：原件 + 复印件
- **费用**：现金或支付宝（约 750 元，价格可能变动）
- **黑色签字笔**（填写表格用）

## 检查流程

![出国健康检查流程](./images/image_1.jpeg)


- 到达医院二层，先在电梯旁的机器取号
- 第二步是当场照相，要露出耳朵眉毛。拍照的人会问你去哪个国家，这个时候还需要提供白底证件照，拍完后给你单子现场填
- 填完单子等待叫号，这一步就是签名和缴费
- 然后就开始体检啦！刷脸进入体检区，护士会指引你并给你一个手环和一件宽大的衣服，先去更衣室换衣服。这里注意女生要把头发盘起，内衣也要脱掉，鞋子换成提供的拖鞋。等于全身上下除了内裤，医院给的衣服和手环，其他的都得存起来！
- 去一个房间先刷手环，医生会问你亲属有无得肺结核的，近期有无感冒发烧，如实回答就好了。
- 终于到了排队等待体检的时候，坐在门外的沙发上等待叫名字进去就行
- 检测完后换回衣服，到咨询台归还手环，告知是当日取结果还是邮寄到付。当日取的话是下午三点到四点出结果，邮寄的话是顺丰，当场手机填，当日就会寄出。我是选择邮寄，同城第二天早上就到了！最后就可以走啦！
