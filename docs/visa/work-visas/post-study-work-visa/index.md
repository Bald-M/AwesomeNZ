# 毕业后工签（Post-study Work Visa）

完成新西兰院校学业后，可申请毕业后工签（Post-study Work Visa），在新西兰合法工作。本文介绍材料准备及申请流程概览。

::: tip
以下信息供参考，具体以移民局官网 [immigration.govt.nz](https://www.immigration.govt.nz/) 最新要求为准。
:::

## 申请时限

- 需在**学生签证到期后 3 个月内**递交申请
- 博士课程毕业生可延至 6 个月内

## 材料清单

| 材料 | 说明 | 详见 |
|------|------|------|
| 护照 | 个人信息页 | — |
| 体检报告 | eMedical 体检（胸片 + 可能需全套体检） | [体检](/visa/work-visas/post-study-work-visa/medical-examination/) |
| Completion Letter | 学校出具的课程完成证明 | [Completion Letter](/visa/work-visas/post-study-work-visa/completion-letter/) |
| 成绩单 | Academic Transcript | — |
| 照片 | 符合移民局要求的证件照 | — |
| Visa 卡 | 支付签证费 | — |

## 申请流程概览

1. **体检**：前往移民局指定体检机构完成 eMedical 体检
2. **获取 Completion Letter**：向学校申请课程完成证明
3. **在线申请**：登录移民局系统，填写并提交工签申请

详见 [移民局在线申请流程](/visa/work-visas/post-study-work-visa/immigration-application/)。
