# 学生签证

本节介绍新西兰学生签证（Student Visa）的申请流程及材料准备。

## 已覆盖签证类型

| 类型 | 说明 | 链接 |
|------|------|------|
| [学生签证（Student Visa）](/visa/student-visas/student-visa/) | 赴新西兰留学签证 | 在线申请 |

## 申请入口

新西兰移民局官网：[immigration.govt.nz](https://www.immigration.govt.nz/)

- 进入「Explore and select a visa」
- 选择「Student visa」
- 使用 **RealMe** 账号登录后在线申请

## 相关指南

申请学生签证前，通常还需准备：

- [出国体检 - 胸片](/pre-departure-medical/)：胸片有效期 36 个月
- [无犯罪记录证明](/criminal-record/)：中国 / 新西兰
- [银行流水](/bank-statement/)：存款证明
