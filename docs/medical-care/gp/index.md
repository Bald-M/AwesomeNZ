# GP（家庭医生）

在新西兰，日常就医通常先找 **GP（General Practitioner，家庭医生）**。GP 是全科医生，负责常见病诊治、开具处方、转诊专科等。

::: tip
建议抵纽后尽早注册一家离家或学校较近的诊所作为你的 GP。
:::

## 注册 GP

- 选择离家或学校较近的诊所，携带护照、学生证等材料前往注册
- 注册后该诊所为你的「家庭医生」，日常不适先联系他们预约

## 预约与就诊

- **预约方式**：电话或该诊所的在线系统
- **就诊流程**：按预约时间到诊所 → 前台签到 → 候诊 → 进诊室就诊

## 费用

- 公民/居民：部分项目可补贴
- 持学生签证：通常需自费，若有学生保险可报销（参见 [看病报销](/medical-care/reimbursement/)）
