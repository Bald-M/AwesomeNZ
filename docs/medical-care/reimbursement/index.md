# 看病报销

在新西兰看病后，若购买了相应保险，可按规定申请报销。常见包括学生保险、旅游保险等。

::: tip
报销前请确认你的保险类型及保障范围，保留好收据、诊断书等材料。
:::

## 通用报销流程（简要）

1. **就医时**：向诊所/医院说明你有保险，部分机构可直接与保险公司结算
2. **自费后报销**：若需先自费，保留 Tax Invoice、Consultation Note 等材料
3. **提交申请**：按保险公司要求通过官网、邮件或 App 提交
4. **等待审核**：通常数日至数周内到账
