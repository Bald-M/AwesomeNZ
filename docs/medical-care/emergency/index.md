# 急诊

遇到紧急健康问题或非工作时间身体不适时，可前往急诊、After Hours 诊所或 Urgent Care。

::: warning
若遇严重紧急情况（如生命危险、严重外伤），请拨打 **111** 呼叫救护车。
:::

## 何时去急诊

- 生命危险、严重外伤、突发剧烈症状等
- 不确定时可拨打 **Healthline**：0800 611 116（24 小时免费健康咨询）

## 常见急诊渠道

| 类型 | 适用场景 | 说明 |
|------|----------|------|
| **医院急诊（ED）** | 严重紧急 | 如 Auckland City Hospital 等，可能等候较久 |
| **After Hours 诊所** | 非工作时间不适 | 部分药店、诊所提供晚间/周末服务 |
| **Urgent Care** | 急但不危及生命 | 骨折、割伤等需尽快处理的情况 |

## 如何贡献其他机构

若你有其他医院/诊所的就医经历，欢迎贡献：

4. 提交 Pull Request
