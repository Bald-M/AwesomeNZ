# 看病

在新西兰看病通常先找 **GP（家庭医生）**，紧急情况可去 **急诊** 或 **After Hours** 诊所。

::: tip
新西兰医疗体系与国内不同，建议抵纽后尽早注册 GP，了解附近急诊及 After Hours 服务位置。
:::

## 紧急求助

- **111**：救护车、消防、警察（严重紧急）
- **Healthline 0800 611 116**：24 小时免费健康咨询（不确定时拨打）
